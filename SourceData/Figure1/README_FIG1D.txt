DataPref is the matrix containing the behaviour for the preference-based task. 
DatePerc is the matrix containing the behaviour for the perceptual task. 

They are both organised as follows: 
- each line is a trial 
- Columns: 
	1. Subject number
	2. Performance (0 = incorrect/ 1 = correct)
	3. Chosen Value
	4. Unchosen Value
	5. Chosen Luminance
	6. Unchosen Luminance
	7. Choice (+1=top/-1=bottom)
 Columns 3 to 7 have been z-scored before entering in the logistic regression.
